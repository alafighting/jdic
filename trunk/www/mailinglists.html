Forums and Mailing Lists

JDIC has a set of forums and mailing lists.

The JDIC forum is a subforum under the Java Desktop Technologies forum. This forum is for general discussion for end users and for developers to discuss project development issues and requirements.

JDIC has four mailing lists:

announce Announcements of new releases and other major project events.
jdic The main list for general discussion for the projects end users and developers to discuss project project development issues and requirements.
cvs Automated notification of file version control changes in the project and discussion of those changes.
issues Automated notifications and discussions of the projects reported issues and changes made to issues.
