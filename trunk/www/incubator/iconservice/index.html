Icon Service

Introduction

Ever wonder how to translate the String returned by Association.getIconFileName() into an Icon? Just how are the following icon specifications translated into Icons? 

C:\Program Files\Common Files\ACD Systems\PlugIns\IDE_ACDStd.apl,1
D:/WINNT/Installer/{00010409-78E1-11D2-B60F-006097C998E7}/wordicon.exe,1
This JDIC Incubator project makes your life simple by introducing the IconService which will create Icons from an icon specifications. This service works for both Windows and Gnome.
