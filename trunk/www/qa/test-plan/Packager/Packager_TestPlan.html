3. Test Procedure

Summary: The testing work includes generating packages by using test .jnlp files, installing packages and uninstalling packages.

3.1.Setup environment

  • Make sure the the j2sdk1.5.0 or higher version is installed on the system

  • Make sure the MS SDK is installed on the system ( this setting is only for Windows platform)

  • Add the jdic.jar(to be tested) to the environment variable CLASSPATH

  • Add the javaws.jar and deploy.jar to the enviroment variable PATH

3.2 Generate packages

  • Windows: jnlp2msi [options] -msd <MS SDK dir> <jnlp file path>

  • Linux: jnlp2rpm [options] <jnlp file path>

  • Solaris (for Sparc and x86): jnlp2pkg [options] <jnlp file path>

3.3 Install packages

  • Windows: double click the .msi file

  • Linux: rpm -i <package name>

  • Soalris (for Sparc and x86): pkgadd -d <package path> <package name>

Start the applications to check if the applications installed correctly.

3.4 Uninstall packages

  • Windows: double click the .msi file

  • Linux: rpm -e <package name>

  • Solaris (for Sparc and x86): pkgrm <package name>
