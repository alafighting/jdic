List of RFE (Request For Enhancement)

Below is a list of RFEs (without order of priority) we received that fit into the scope of JDIC: desktop integration, cross-platform APIs/facilites. Those RFEs that are already addressed in the project are not listed. 

These RFEs are collected from the project forum, issue tracker, feedbacks and other places. We'll keep updating this page to add more RFEs, and update the "Status" column. Contributions to any of the RFEs are very welcome and could be held as an incubator project in the JDIC "Incubator Projects" space. If you'd like to hold an incubator project, please drop an email to owner@jdic.dev.java.net (alias for the project owners). If you already have something available, you can also contact us, so that we can work together to achieve the same goals.
