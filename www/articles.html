Articles and Blogs on JDIC

This page collects those articles and blogs around JDIC, which provide valuable usage guidelines and evaluation from the user's perspective. If you write or find such articles or blogs somewhere, please leave owner@jdic.dev.java.net (the alias for the project owners) an email, we'll add the link to this page.

The JDIC Overview article, Joshua Marinacci's two-part sketchbook Look, feel, and behave like a native application and Save your screen, with Java!, and George Zhang's blog Where's JDIC Going? give an overall introduction or update of the project.

  • Tech Tip: Communicating with Native Applications Using JDIC
    The May 5 issue of Core Java Technologies Tech Tips shows how to load a web page in a JFrame using the native browser hooks provided by the JDIC API. (May 09, 2005) 
  • Article: JDIC Tray Icons
    In OCI's article JDIC Tray Icons Erik M. Burke leads you through an example of "a dummy weather forecast service that shows current weather conditions as an animated GIF on the system tray. Balloon help alerts you when a new forecast is available and you can click the icon to view a detailed forecast in a Swing JFrame." (May, 2005)
  • Article: JDIC - Java Desktop Componets
    This page by Jean-Marc Autexier gives an overview about the JDIC project, which contains some article links around JDIC. (April, 2005)
  • Blog: JDIC Features in Mustang
    "In the recent months, the JDIC team has been working closely with the J2SE team to incorporate some of the exciting features from JDIC into Mustang. This shows a potential reward in contributing to the JDIC project: one day the contributed feature may become integrated into J2SE!" — George Zhang (April 24, 2005)
  • Article: Run Java Apps from SysTray 
    This article provides a step-by-step guide to use JDIC Tray Icon API to integrate Java apps into Windows SysTray (April 12, 2005)
  • Blog: Sometimes you only need a little.
    "Joshua talks about the new JDIC Misc API for small but useful features, including alerts, volume, and dock menu support on Mac OS X" — Joshua Marinacci (April 11, 2005) 
  • Article: JDIC(Jdesktop Integration Components)入门 (In Chinese)
    "本文简单介绍了JDIC(JDesktop Integration Components)的用途，以及以实例的形式教你如何使用JDIC调用或者整合本地桌面环境的组件或应用程序。" (November 17, 2004)
  • Blog: Why I can't get excited about JDIC
    Elliott Hughes listed the things JDIC can do and can not do ... (November 07, 2004)
  • Blog: Embed Mozilla or IE as Java canvases
    "Sun's new JDIC looks like a good step forward"  — Rick Jelliffe (June 09, 2004)
