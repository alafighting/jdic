Articles and Blogs on JDIC

This page collects those articles and blogs around JDIC, which provide valuable usage guidelines and evaluation from the user's perspective. If you write or find such articles or blogs somewhere, please leave owner@jdic.dev.java.net (the alias for the project owners) an email, we'll add the link to this page.

The initial JDIC Overview article and George Zhang's Blog give an overall introduction or update of the project.

  • August 09, 2005 - IBM developerWorks - Introducing the JDesktop Integration Components
    "This article introduces the JDesktop Integration Components (JDIC), an open source project that's building components that bridge the gap between native applications and their Java™ counterparts. JDIC's single Java API lets your applications tap into native OS features while maintaining cross-platform support. It currently offers native Web browser (Internet Explorer or Mozilla) support, system tray support, file-extension integration, and other desktop features." And it provides a "wish list" with a couple of features falling into the scope of JDIC.
  • May 09, 2005 - java.sun.com - Tech Tip: Communicating with Native Applications Using JDIC
    The May 5 issue of Core Java Technologies Tech Tips shows how to load a web page in a JFrame using the native browser hooks provided by the JDIC API.
  • May, 2005 - OCI - JDIC Tray Icons
    In OCI's article JDIC Tray Icons Erik M. Burke leads you through an example of "a dummy weather forecast service that shows current weather conditions as an animated GIF on the system tray. Balloon help alerts you when a new forecast is available and you can click the icon to view a detailed forecast in a Swing JFrame."
  • April, 2005 - autexier.de - JDIC - Java Desktop Components
    This page by Jean-Marc Autexier gives an overview about the JDIC project, which contains some article links around JDIC.
  • April 24, 2005 - java.net - JDIC Features in Mustang
    "In the recent months, the JDIC team has been working closely with the J2SE team to incorporate some of the exciting features from JDIC into Mustang. This shows a potential reward in contributing to the JDIC project: one day the contributed feature may become integrated into J2SE!"
  • April 12, 2005 - PCQuest - Run Java Apps from SysTray 
    This article provides a step-by-step guide to use JDIC Tray Icon API to integrate Java apps into Windows SysTray.
  • April 11, 2005 - java.net - Sometimes you only need a little. (blog)
    "Joshua talks about the new JDIC Misc API for small but useful features, including alerts, volume, and dock menu support on Mac OS X".
  • November 20, 2004 - Symphonious - JDIC (blog)
    This blog commented on the two "most" interesting features: tray icon integration and browser integration and mentioned some features missing in the project.

    "本文简单介绍了JDIC(JDesktop Integration Components)的用途，以及以实例的形式教你如何使用JDIC调用或者整合本地桌面环境的组件或应用程序。"
