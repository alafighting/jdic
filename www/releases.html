JDIC Releases

This page gives the release notes for all the major JDIC releases (major.minor.micro) and some sub-releases (major.minor.micro.*). These release notes describe system requirements, installation instructions, what's new, known issues and all the relevant release information. Certain sub-releases may not be listed at this page but downloadable at the Documents & files page, if they included few minor updates/fixes.

These notes are updated when we receive feedback, so please check back for new information. Consult the Documents & files page to download the source and the binary builds for your platform.

Note: The releases with the gray font are the next JDIC releases under construction, which include a target release date and planned new features and bug fixes. The planned release number, release date and new features may change in the final release.

Log of cvs tags for major public releases

 

JDIC 0.8.7 - Released Dec. 17, 2004

  • Download - the release includes the source, and binary builds for Windows/Linux/Solaris.
  • The README file contains installation instructions, system requirements, license terms, and instructions to pull the JDIC source from CVS and build it.
  • See the ChangeLog file for a comprehensive list of fixes and known issues, which includes all of the new features and fixes from 0.8.4 (the initial release).
  • What is new -
    • Desktop Mailer support Mozilla Thunderbird on Windows/Linux/Solaris
    • WebBrowser works fine when browsing https web site.
    • Automatically set JAVA_HOME and java.library.path property for TrayIcon.
    • Environment variables are setted by code automatically, you just need to add the jdic.jar to your classpath.
    • Fix several critical bugs of TrayIcon.
    • Fix several critical bugs of Packager.
    • Remove several unnecessary linked library to build jdic.
    • Refine code for init package.

JDIC 0.8.6.3 - Released Nov. 10, 2004

  • Download - the release includes only the binary for Solaris x86 platform.
  • An update release to JDIC 0.8.6, which fixes below two critical, known issues:
    • The Browser component (API) only works with mozilla 1.4 on Unix (issue#87). 
      With the fix, it works with mozilla 1.4 through 1.8a4 using gtk2 On Solaris x86 platform. 
    • Building the JDIC (in specific, Browser component) source requires building a Mozilla source on Unix (issue#119).
      With the fix, it builds against Gecko SDK and a Mozilla binary, both of which are downloadable from mozilla.org.

JDIC 0.8.6 - Released Sept. 30, 2004

JDIC 0.8.5 - Released Sept. 07, 2004

  • Download - the release includes the source, and binary builds for Windows, Linux and Solaris platforms. There is also a binary for FreeBSD, contributed by Henri Michelon.
  • The README file contains installation instructions, system requirements, license terms, and instructions to pull the JDIC source from CVS and build it.
  • See the ChangeLog file for a comprehensive list of fixes and known issues, which includes all of the new features and fixes from 0.8.4 (the initial release).
  • What's New - outlines some of the new features implemented in this release and important bugfixes.
    • The Browser demo supports "https" protocol.
    • WebBrowser.willOpenURL(URL url) supports URLs with "file" as the prototol.
    • The Browser component supports Java Web Start application. 
    • The Browser component can check the default browser, without requiring to set MOZILLA_FIVE_HOME. 
    • The Browser component supports TitleChange and StatusTextChange event.
    • The Browser component works with mozilla 1.4.1 downloaded from mozilla.org.
    • Support for Gnome 2.0+, without liblinc.so link error for Gnome 2.6.
    • JDIC Packager supports remote JNLP applications via a jnlp URL.
    • JDIC Packager correctly checks javaws path from /etc/mailcap file on Unix.
    • Lots of fixes to JDIC Package for Windows platform.
    • Updated COPYING file.
