Projects Using JDIC

This is a list of projects or products using JDIC. If your project is using JDIC, please drop us an email with your project link, description, and the JDIC features it uses, we will be happy to include it in this list.

  • Eye of Horus - a skinnable java surveillance software
    It uses JDIC's tray icon and browser support.
  • Rommity - provides a rich internet environment to read the latest news feeds and publish their ideas
  • JavaHelp - a full-featured, platform-independent, extensible help system
    It supports using JDIC Browser component as an alternative content viewer. 
  • Columba - an Email Client written in Java, featuring a user-friendly graphical interface with wizards and internationalization
    support
    It incorporates the JDIC TrayIcon support.
  • JFTP - a secure, graphical FTP client software
    It uses JDIC Desktop API to open, edit, print and e-mail local and/or remote files using associated applications.
  • TradeFrame - a financial analysis tool written in Java/Swing
    "... uses the following features of JDIC:
    - Web-Browser integration (the Home tab of the main tabbed pane)
    - Tray Icon (which is animated during computations)
    - Launch of default mailer (the contact us menu item)...."
  • NVoy RSS Ticker - an RSS ticker application
    "... uses JDIC components, especially browser and tray icon components ..."
  • JADETray - an application to manage a JADE (Java Agent Development Framework) agent container through a more friendly user interface It uses JDIC Tray Icon API.
  • Xito - the Xito Application Manager, designed for end users, provides a new way to launch and manage Java Internet Content
