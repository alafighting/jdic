Project Feedback

The JDIC project (JDIC) was open sourced at JavaOne 04 (June, 2004). There have been thousands of downloads and lots of contributions. If you are using JDIC, or plan to use it in your projects, please let us know of the experience you have with the project. The feedbacks are very helpful to the project, so that we know better what are the real requirements and how we can improve it. 

This page is for the usage scenarios of JDIC in real projects. For bug reports and general technical discussions, please refer to below links:

  • Forum - forum for the project.
  • Mailing List - contact relevant groups with issues.
  • Issue Tracker - report bugs.

The submitted feedback will be emailed to owner@jdic.dev.java.net automatically, which is the alias for the project owners. Now it includes George Zhang, Joey Shen, Thorsten Laux and Paul Huang. Then it will be added to the following feedback list. If you do not want it posted in this page, DO REMEMBER to tell us. Thank you very much for your feedback !
