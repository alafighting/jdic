Project Feedback

The JDIC project was open sourced at JavaOne'04 (July, 2004). There have been thousands of downloads and lots of contributions. If you are using JDIC, or plan to use it in your projects, please let us know your experience with the project. The feedbacks will help us know what are the real requirements and how we can improve it. 

This page is for the usage scenarios of JDIC in real projects. For bug reports and general technical discussions, please refer to below links:

  • Forum - forum of the project.
  • Mailing List - contact relevant groups with issues.
  • Issue Tracker - report bugs.

You can send the feedbacks to owner@jdic.dev.java.net, which is the alias for the project owners. Then the feedbacks will be added to below feedback list. If you do not want them posted in this page, please REMEMBER to tell us. Thanks for any feedbacks !

"I choose the JDIC systray project ...... for several reasons:

1) Small, simple API - did exactly what it said on the box
2) I love being able to use animated GIFs for the systray - something that I couldn't with S4J which only used .ico's (at the time, I haven't checked recently).
3) It used Java LaF so I could ue JGoodies Looks to give my whole app the same LaF, again I couldn;t do this with S4J...

The project is an application to manage a JADE (Java Agent Development Framework - http://jade.tilab.com) agent container through a more friendly user interface. Agent containers are typically designed to be long running, hence they shouldn't take up too much screen space, which is why a tray icon/menu is the most logical choice. The user can manage all aspects of the container from the desktop using the simple menu interface to start and stop agents as well as shutdown the container. ......"
